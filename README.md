# Hand-Gesture-Controll-Led
https://drive.google.com/file/d/1flKyOcyNspNzacBt7nuZR5Febmx_REPf/view?usp=drivesdk
Developed a computer vision–based system using Python and MediaPipe to detect hands and recognize finger gestures in real time via a webcam.
Orchestrated the bi-directional communication between Arduino Uno and Python library numpy achieving an average latency reduction of 45 milliseconds for real-time IoT device interaction.
Implemented gesture-to-action mapping where specific raised fingers control corresponding LEDs on the Arduino, demonstrating real-time IoT interaction.
